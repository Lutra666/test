# Web Profile Templates

这个仓库包含两套可直接预览与复用的静态网页模板：

- 个人介绍页（`网页设计/个人介绍`）
- 企业介绍页（`网页设计/企业介绍`）

目标是快速搭建「可展示、可维护、可迭代」的介绍型网页。

## 项目结构

```text
.
├─ 网页设计/
│  ├─ 个人介绍/
│  │  ├─ index.html
│  │  └─ avatar.jpg
│  └─ 企业介绍/
│     ├─ index.html
│     ├─ styles.css
│     ├─ render.js
│     ├─ company-data.js
│     └─ README.md
└─ 网页项目模板介绍.md
```

## 快速预览

直接在本地双击以下文件：

- `网页设计/个人介绍/index.html`
- `网页设计/企业介绍/index.html`

不依赖打包工具，可离线预览。

## 如何修改内容

### 个人介绍页

主要在 `网页设计/个人介绍/index.html` 中修改：

- 姓名、简介、经历、教育、联系方式
- 头像文件默认是 `avatar.jpg`

### 企业介绍页

优先改 `网页设计/企业介绍/company-data.js`：

- 公司名称、定位语、简介
- 核心数据、核心业务
- 时间线、联系方式、页脚

样式与交互逻辑分别在：

- `styles.css`：视觉样式
- `render.js`：数据渲染与主题切换逻辑

## 设计与交互规范（当前版本）

- 风格：平面、克制、黑灰中性
- 主题：支持亮色/暗色切换，并记忆上次选择
- 动效：保留必要过渡，避免炫光与过度装饰
- 响应式：适配桌面和移动端
