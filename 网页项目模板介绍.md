# 网页项目模板介绍

这是一个面向个人展示与企业展示场景的前端静态模板仓库，目标是用较低成本快速搭建可上线、可维护、可扩展的介绍类网页。

## 项目定位

- 适合做个人主页、工作室展示、企业介绍页
- 不依赖复杂框架，直接打开 `html` 即可预览
- 保留现代交互（主题切换、动效过渡）并兼顾简洁风格

## 当前结构

```text
网页设计/
  个人介绍/
    index.html
    avatar.jpg
  企业介绍/
    index.html
    styles.css
    render.js
    company-data.js
    README.md
```

## 模板特点

1. 双主题切换  
支持亮色/暗色模式切换，使用本地存储记住用户偏好，刷新后可保持主题状态。

2. 平面高级风格  
整体视觉已收敛为简洁层次，减少炫光与过度装饰，更适合正式展示。

3. 数据与视图分离  
企业介绍页采用 `company-data.js` 管理内容，非技术用户也能直接改文案。

4. 响应式布局  
已适配桌面与移动端，常见宽度下都可正常阅读。

5. 可持续扩展  
后续可继续加入：作品集模块、服务报价模块、留言表单、部署脚本等。

## 快速使用

1. 进入对应目录（个人介绍或企业介绍）
2. 修改文案与联系方式
3. 替换头像或品牌素材
4. 双击 `index.html` 本地预览
5. 完成后提交并推送到 GitHub

## 推荐优化方向

- 为个人介绍页补充真实经历、技能与项目案例
- 为企业介绍页增加案例展示与客户评价
- 增加统一的仓库级 `README.md`，方便协作与交接

---

如果你希望，这个模板还可以进一步抽象成「一套配置，多页共用」的形式，让个人页和企业页共用同一套主题和组件规范。
