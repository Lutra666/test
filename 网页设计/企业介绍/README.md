# 企业介绍页面

这是一个可直接打开预览的企业介绍模板。

## 如何使用

1. 打开 `company-data.js`
2. 按注释填写你们公司的真实信息
3. 双击 `index.html` 预览页面

## 文件说明

- `index.html` 页面结构
- `styles.css` 页面样式
- `company-data.js` 企业信息配置（你主要改这个）
- `render.js` 渲染脚本

## 你需要准备的信息

- 公司名称（中英文可选）
- 一句话定位（比如“专注智能制造的数字化解决方案”）
- 核心介绍文案（1-2 段）
- 核心数据（成立时间、团队规模、服务客户数等）
- 核心业务（3-6 条）
- 发展历程（年份 + 事件）
- 联系方式（电话、邮箱、地址）
